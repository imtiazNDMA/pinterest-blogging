## SAFETY — flag prominently for the writer (this is a BUILD post)
Hedge, never fabricate. Required:
- **Power & hand tools:** wear eye protection; follow each tool's own instructions; this post is general inspiration, not a substitute for tool manuals.
- **Wood for ground contact / food-adjacent:** explain pressure-treated vs. untreated; caution on using pressure-treated where skin/food contact is frequent.
- **Reclaimed pallets:** use **heat-treated (HT-stamped)** pallets only; **avoid chemically treated (MB-stamped)** pallets; inspect for spills/stains.
- **Sanding & splinters:** sand cut edges; mind dust (mask).
- **Sealant/stain:** ventilate; follow manufacturer directions; let cure.
- **Stability & weight:** finished pieces must be stable — anchor/weight lightweight builds against wind; don't overload built seating. **No fabricated load claims** (e.g., "I built this and it held 300 lbs"). Use `[PERSONAL TIP]` only for genuinely user-supplied experience.
