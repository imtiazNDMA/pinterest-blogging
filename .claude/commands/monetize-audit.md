Run a monetization audit. Scope: `$ARGUMENTS` (empty = whole site).

Dispatch the **monetization-strategist** agent to:
1. **Affiliate coverage:** review existing/planned posts (tracker + `Content/`) for monetization gaps —
   informational clusters with no buyer post, buyer posts missing better garden-affiliate programs vs Amazon.
   Update `Growth Strategy/affiliate-programs.md`.
2. **Product opportunities:** suggest compliant "best X" / comparison posts worth adding, with the affiliate
   source per product (qualitative, NO prices).
3. **Ad-network readiness:** audit against AdSense, Mediavine Journey (~1k sessions + Grow plugin), and
   Raptive (~25k pageviews, tier-1 traffic). Output a gap checklist + which milestone unlocks each.
   Update `Growth Strategy/ad-network-readiness.md`.
4. **Compliance spot-check:** confirm disclosures + no price/stock claims across monetized posts.

Report the top revenue-leverage moves first, each with a concrete next action. Re-verify program rates
before relying — they change.
