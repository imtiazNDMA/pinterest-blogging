# ⚙️ Content Workflow

← [[Home]] · [[Content Tracker]] · [[Pinterest Strategy]]

How a post goes from idea → published + pinned. The heavy lifting (drafting, QA, pins) runs in the **production
studio** (the `.claude/` system); this vault tracks status and the human steps.

## The pipeline
```
1. PICK   → take the top idea from [[Content Backlog & Ideas]]
2. DRAFT  → studio writes the post (brief → draft → monetize → QA → Ready)
3. FILL   → YOU add owner inputs: real photos, [PERSONAL TIP], real affiliate links, replace {{tokens}}
4. PUBLISH→ paste to WordPress, set category/meta/featured image, hit publish
5. PIN    → studio builds the pin batch; YOU generate images + schedule (see [[Pinterest Strategy]])
6. LOG    → update [[Content Tracker]] + [[Pin Tracker]]
7. REVIEW → monthly, check what's winning ([[Monthly Review]]) → feed back into ideas
```

## Owner inputs (the part only you can do)
- **Photos** — real or AI-generated styled images (the studio gives design briefs + AI prompts).
- **[PERSONAL TIP]** — one genuine first-person line per post (never faked — it's the honesty anchor).
- **Affiliate links** — real tagged URLs; FTC disclosure above the first one.
- **Tokens** — replace `{{BLOG_NAME}}` / `{{DOMAIN}}` / `{{AMAZON_TAG}}`.

## Quality bar (the studio enforces; know it)
- No AI slop, no em-dashes/emojis/clickbait, scannable formatting.
- One primary keyword, intent fully satisfied, 3+ internal links.
- Plant/safety claims verified. No Amazon prices/"cheapest"/stock.
- Disclosure above first affiliate link.

## WordPress publish checklist
- [ ] Title + slug (no year) · meta description (140–160 chars)
- [ ] Primary category ([[WordPress Categories & Tags]]) · 1–3 tags
- [ ] Featured image (2:3 pin works) + alt text on every image
- [ ] 3+ internal links + link up to the pillar
- [ ] FTC disclosure block · affiliate links tagged
- [ ] Table of contents (long posts) · mobile preview
- [ ] Submit URL in Google Search Console

## Repurpose each post into
- 5 fresh pins (multi-angle) → [[Pin Tracker]]
- 1 email to the list (once it exists)
- Internal links from/to siblings + pillar
