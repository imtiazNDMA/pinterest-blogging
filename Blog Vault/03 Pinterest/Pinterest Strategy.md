# 📌 Pinterest Strategy

← [[Home]] · [[Pin Tracker]] · [[Boards Plan]] · [[Pinterest Analytics]]

Pinterest is a **visual search engine**, not social media. Treat it as SEO. It's your **primary traffic source** —
the blog is where you monetize the clicks.

## The rules that matter (2026)
- **Fresh pins win.** A fresh pin = a new image/design, not a re-share. Every post gets multiple distinct pins over time.
- **Strict 2:3 / 1000×1500 px.** Bold, thumbnail-legible text overlay. On-brand and consistent.
- **Pins must match their destination** — no clickbait/bait. The post must deliver the promise.
- **Label AI-generated pins.** Home decor is on Pinterest's 2026 GenAI "see less" list → label honestly, keep renders authentic.
- **Keyword everything:** pin title, description, board name, image filename.
- **No emojis, no hashtag spam.** A few relevant keywords in the description is enough.

## Cadence (ramp gradually — don't spike a new account)
| Stage | Pins/day |
|---|---|
| New account (month 1) | 1–3, spaced |
| Warmed up | 5–10, spaced |
- Best posting window ≈ **8–11 PM** in your audience's timezone (validate with your own analytics).
- A few Idea/video pins per week for algorithm trust (reach, not direct clicks).

## Workflow
1. Studio builds the pin batch (5 angles) → `Content/pins/<slug>-pins.md`.
2. Generate images from the AI prompts (or design in Canva) — see the studio's pin design system.
3. **QA:** 2:3, authentic, no garbled text/logos/prices, on-brand, labeled AI.
4. Schedule via the **Pinterest native scheduler**, spaced, evening slots.
5. Log in [[Pin Tracker]]; update status as they publish.

## Multi-angle pin formula (per post)
Each post → 5 fresh pins, different angle = different design:
**Aesthetic hero · List/number grid · Before/after · Comparison/curiosity · Product/category.**

## Pin-keyword retargeting (important)
Several blog keywords don't exist as Pinterest interests — pins must use the terms that DO:
- Pillar → "balcony ideas" / "small balcony design" · Cozy → "cozy balcony" · Makeover → "balcony makeover" ·
  Renter → "small balcony decor". (Full map in the studio validation file.)
