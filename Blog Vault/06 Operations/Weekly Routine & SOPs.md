# 🗓️ Weekly Routine & SOPs

← [[Home]] · [[Content Calendar]] · [[Content Workflow]] · [[Dashboard]]

Your ~20 hrs/week, structured so the engine keeps moving without burning out. Batch similar work; protect a
weekly rhythm.

## Weekly rhythm (~20 hrs)
| Day | Focus | Hrs |
|---|---|---|
| **Mon** | Plan the week ([[Dashboard]] → This week); review last week's pins | 1 |
| **Tue** | Write / fill owner inputs on 1–2 posts | 4 |
| **Wed** | Write / publish; set categories, meta, internal links | 4 |
| **Thu** | Pin design — generate + QA a week of images | 4 |
| **Fri** | Schedule pins; Pinterest housekeeping (boards, descriptions) | 3 |
| **Sat** | Buffer / SEO tweaks / learning / outreach | 2–3 |
| **Sun** | Off (or light idea capture in [[Content Backlog & Ideas]]) | — |

> Adjust freely — the point is **batching** (write-batch, pin-batch) and a repeatable cadence, not these exact days.

## SOP — Launch (do once)
1. Pick brand name → buy `{{DOMAIN}}` → replace all `{{tokens}}`.
2. Install WordPress + fast theme; configure permalinks to `/%postname%/`.
3. Add legal/trust pages: About, Contact, Privacy, **Affiliate Disclosure**, Terms ([[Site Structure]]).
4. Install plugins: SEO, caching, image optimization, ToC.
5. Connect **GA4** + **Search Console**; submit sitemap.
6. Create Pinterest Business account; claim site; enable Rich Pins; build 5 boards ([[Boards Plan]]).
7. Publish first 1–2 posts; ship first pin batch.
8. Apply to Amazon Associates.

## SOP — Publish a post
→ Follow the checklist in [[Content Workflow]] (owner inputs → WordPress → category/meta → internal links → disclosure → GSC submit).

## SOP — Ship a pin batch
1. Open the post's pin briefs (studio `Content/pins/<slug>-pins.md`).
2. Generate images from the AI prompts (or Canva); QA each (2:3, authentic, no garbled text/prices/logos, on-brand).
3. Label AI; schedule spaced via Pinterest native scheduler (evening slots).
4. Log in [[Pin Tracker]].

## SOP — Monthly review
→ [[Monthly Review]] on the 1st: pull numbers, log wins, set 3 priorities, update [[Goals & Milestones]].

## Anti-burnout rules
- One small ship a day beats a heroic weekend then a week off.
- Done > perfect. A live "good" post earns; a perfect draft doesn't.
- Protect one day off. The compounding works while you rest.
