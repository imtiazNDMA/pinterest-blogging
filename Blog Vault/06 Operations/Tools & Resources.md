The stack. **Bootstrap rule: free tools unless a cost is clearly worth it.**

## Core stack
| Need | Tool | Cost | Notes |
|---|---|---|---|
| Hosting + CMS | WordPress (self-hosted) | $ | The site |
| Theme | Kadence / GeneratePress (free) | Free | Fast, lightweight |
| SEO | Rank Math / Yoast (free) | Free | Titles, meta, sitemap, schema |
| Analytics | Google Analytics 4 | Free | Sessions, pageviews |
| Search | Google Search Console | Free | Queries, indexing |
| Pin design | Canva (free) | Free | Text overlays on AI image plates |
| AI images | ChatGPT / Gemini / Midjourney | $/free | Styled scenes (see studio prompts) |
| Pin scheduling | Pinterest native scheduler | Free | No third-party needed early |
| Keyword data | PinClicks / Pinterest Trends | $/free | Validation exports |
| Writing/QA | The production studio (`.claude/`) | — | Draft → QA → pins |

## Vault management (this Obsidian vault)
- **Obsidian** (free) — open this folder as a vault.
- Optional plugins: **Dataview** (live tables from frontmatter), **Calendar**, **Kanban** (board view of [[Content Tracker]]).
- Use **Graph view** to see how everything connects; **Quick switcher** (Ctrl/Cmd-O) to jump anywhere.
